不等式集锦

基础数学补充
  • 数学

常见不等式的总结

均值不等式PLUS

$$
\frac{2}{\frac{1}{a}+\frac{1}{b}} \leq \sqrt{a b} \leq \frac{a+b}{2} \leq \sqrt{\frac{a^{2}+b^{2}}{2}}
$$

　　这四个被称为均值不等式，其中a，b均为正数。调和平均 < 几何平均 < 算数平均 < 平方平均。但实际上这四个不等式是可以无限延伸下去的。

第一步，给他们变形：
$$
\left(\frac{a^{-1}+b^{-1}}{2}\right)^{\frac{1}{-1}} \leq \sqrt{a b} \leq \quad\left(\frac{a^{1}+b^{1}}{2}\right)^{\frac{1}{1}} \leq \left(\frac{a^{2}+b^{2}}{2}\right)^{\frac{1}{2}}
$$

　　显然，这里我们会发现调和均值、算数均值、平方均值都有相同的形式，于是考虑到，可以引入函数。

第二步，引入函数：
$$
f(n)=\left(\frac{a^{n}+b^{n}}{2}\right)^{\frac{1}{n}}
$$
　　只要证明$f(n)$ 是一个递增的数列，或者$f(x)$ 单调递增即可。

第三步，证明函数单调递增

　　见下面琴生不等式，举例，Q3。

第四步，处理$\sqrt{a b}$

　　在第一步变型中，可以看到$\sqrt{a b}$和其他三项的形式是不一样的，其他三项分别是$f(-1),f(1),f(2)$。实际上$\sqrt{a b}$是$\lim _{x \rightarrow 0} f(x)$，证明如下：
$$
\begin{aligned}\lim _{x \rightarrow 0} f(x) &=\lim _{x \rightarrow 0}\left(\frac{a^{x}+b^{x}}{2}\right)^{\frac{1}{x}} \&=\exp \left[\lim _{x \rightarrow 0} \frac{\ln \left(\frac{a^{x}+b^{x}}{2}\right)}{x}\right] \&=\exp \left[\lim _{x \rightarrow 0} \frac{\frac{a^{x}+b^{x}}{2}-1}{x}\right] \&=\exp \left[\lim _{x \rightarrow 0} \frac{a^{x}-1+b^{x}-1}{2 x}\right] \&=\exp \left[\lim _{x \rightarrow 0} \frac{x \ln a+x \ln b}{2 x}\right]=\sqrt{a b}\end{aligned}
$$
　　最后得出结论，对于函数$f(x)=\left(\frac{a^{x}+b^{x}}{2}\right)^{\frac{1}{x}}$是一个单调递增的函数，其中均值不等式的表示方法为：
$$
f(-1)<\lim _{x \rightarrow 0} f(x)<f(1)<f(2)<…<f(n)​
$$

　　均值不等式还有一个PLUS版本：
$$
\frac{n}{\sum_{i=1}^{n} \frac{1}{a_{i}}} \leq \sqrt[n]{\prod_{i=1}^{n} a_{i}} \leq \frac{1}{n} \sum_{i=1}^{n} a_{i} \leq \sqrt{\frac{1}{n} \sum_{i=1}^{n} a_{i}^{2}}
$$

举例

$$
f(x)=\sqrt{\sin x}+\sqrt{\cos x} \quad x \in\left[0, \frac{\pi}{2}\right] \quad 求f(x)的最大值
$$

　　这种题目本质上是，已知$a^{4}+b^{4}=1$，求 a + b 的最大值，可以很轻松的用均值不等式PLUS解决：
$$
\begin{array}{c}\left(\frac{a+b}{2}\right)^{\frac{1}{1}} \leqslant\left(\frac{a^{2}+b^{2}}{2}\right)^{\frac{1}{2}} \leqslant \cdots \leqslant\left(\frac{a^{4}+b^{4}}{2}\right)^{\frac{1}{4}}=\left(\frac{1}{2}\right)^{\frac{1}{4}} \\therefore a+b \leqslant 2 \cdot\left(\frac{1}{2}\right)^{\frac{1}{4}}\end{array}
$$

绝对值三角不等式

绝对值放缩，有界，不等式问题用的非常多
$$
\begin{array}{l}|| x|-| y | \leqslant|x \pm y| \leqslant|x|+|y|, \quad \forall x, y \in R .\
\text { 当 } x y \leqslant 0 \text { 时, }|x|-| y| \leqslant|x+y| \text { 取等号， }|x-y| \leqslant|x|+|y| \text { 取等号; }\
\text { 当 } x y \geqslant 0 \text { 时, }|x|-| y| \leqslant|x-y| \text { 取等号, }|x+y| \leqslant|x|+|y| \text { 取等号. }
\end{array}
$$

举例

Q1
$$
f(x, y)=\left{\begin{array}{l}
\frac{x^{3}+y^{3}}{x^{2}+y^{2}}, x^{2}+y^{2} \neq 0 \
0 \quad, x^{2}+y^{2}=0
\end{array}, \text { 判断 } f(x, y) \text { 在 }(0,0)\right. \text { 是否连续. }
$$

$$
\left|\frac{x^{3}+y^{3}}{x^{2}+y^{2}}\right| \leqslant \frac{\left|x^{3}\right|}{x^{2}+y^{2}}+\frac{\left|y^{3}\right|}{x^{2}+y^{2}}=|x| \cdot \frac{x^{2}}{x^{2}+y^{2}}+|y| \cdot \frac{y^{2}}{x^{2}+y^{2}} \leqslant|x|+|y|
$$

琴生不等式

琴生不等式是在高数接触的不等式中，第一个让我感觉到眩晕的不等式， ╮（╯＿╰）╭

琴生不等式是函数凹凸性的一个推论，主要解决的问题是函数值的平均数以及平均数的函数值的问题。

举例

Q1

已知A,B,C是锐角△ABC的三个内角，求$\tan \frac{A}{2}+\tan \frac{B}{2}+\tan \frac{C}{2}$的最小值。

分析：已知A+B+C的值，求$f(A)+f(B)+f(C)$的值，这显然是利用琴生不等式比较方便：
$$
\begin{array}{l}因为A, B, C>0,且 f(x)=\tan 在x \in\left(0, \frac{\pi}{2}\right)上是单调递增的下凸函数； \有琴生不等式可知，有\frac{f\left(\frac{A}{2}\right)+f\left(\frac{B}{2}\right)+f\left(\frac{C}{2}\right)}{3} \geqslant f\left(\frac{A+B+C}{6}\right)\所以，\tan \frac{A}{2}+\tan \frac{B}{2}+\tan \frac{C}{2}\geqslant \sqrt{3}\当且仅当A = B = C的时候，等号成立\end{array}
$$

Q2

若实数a，b满足 a + 2b = 12，则$2^{a}+2^{b+1}$的最小值是多少？

分析：表面上来看这道题没啥思路，但条件和问题可以改成，已知 a + b + b = 12，求$2^{a}+2^{b}+2^{b}$的最小值，这样就和上道题一样，已知A+B+C的值，求$f(A)+f(B)+f(C)$的值，这显然是利用琴生不等式。
$$
\begin{array}构构造函数f(x)=2^{x}，显然该函数是单调递增下凸函数； \有琴生不等式可知，对于实数a，b有\frac{f(a)+f(b)+f(b)}{3} \geqslant f\left(\frac{a+b+b}{3}\right) \\Rightarrow \frac{2^{a}+2^{b}+2^{b}}{3} \geq 2^{\frac{a+b+b}{3}} \\Rightarrow 2^{a}+2^{b+1} \geqslant 3 \cdot 2^{\frac{a+2 b}{3}}=3 \cdot 2^{4}\end{array}
$$

Q3
$$
\text { 求证: } f(x)=\left(\frac{a^{x}+b^{x}}{2}\right)^{\frac{1}{x}} \text { 在 } x \in R \text { 上是一个增函数. }\left(a, b \in R^{+}\right) \text {. }
$$
证明如下：
$$
\begin{array}{l}不妨令\alpha>\beta,则只需证明 f(\alpha)>f(\beta)即可; \只需证明\left(\frac{a^{\alpha}+b^{\alpha}}{2}\right)^{\frac{1}{\alpha}}>\left(\frac{a^{\beta}+b^{\beta}}{2}\right)^{\frac{1}{\beta}}即可; \构造函数g(x)=x^{\frac{\alpha}{\beta}} ;\当\alpha>\beta>0时，g(x)为单调递增下凸函数， \由琴生不等式可知，有x_{1}, x_{2} \in R, \frac{g\left(x_{1}\right)+g(x_2)}{2} \geqslant g\left(\frac{x_{1}+x_{2}}{2}\right) \Rightarrow \frac{x_{1}^{\frac{\alpha}{\beta}}+x_{2}^{\frac{\alpha}{\beta}}}{2} \geqslant\left(\frac{x_{1}+x_{2}}{2}\right)^{\frac{\alpha}{\beta}} \令x_{1}=a^{\beta}, x_{2}=b^{\beta}, \quad \frac{a^{\alpha}+b^{\alpha}}{2} \geqslant\left(\frac{a^{\beta}+b^{\beta}}{2}\right)^{\frac{\alpha}{\beta}} \Rightarrow\left(\frac{a^{\alpha}+b^{\alpha}}{2}\right)^{\frac{1}{\alpha}}\geqslant\left(\frac{a^{\beta}+b^{\beta}}{2}\right)^{\frac{1}{\beta}},即证。\当\alpha>0>\beta,以及当0>\alpha>\beta的情况基本类似，免证。 \综上，该函数是一个单调递增的函数。\end{array}
$$

柯西不等式

$$
\left(a_{1}^{2}+a_{2}^{2}+\cdots+a_{n}^{2}\right)\left(b_{1}^{2}+b_{2}^{2}+\cdots+b_{n}^{2}\right) \geqslant\left(a_{1} b_{1}+a_{2} b_{2}+\cdots+a_{n} b_{n}\right)^{2}
$$

　　这个不等式的证明非常地巧妙且简单，只需要构造两个向量即可：
$$
\begin{array}{l}令
\vec{a}=\left(a_{1}, a_{2}, …, a_{n}\right), \vec{b}=\left(b_{1}, b_{2}, …, b_{n}\right) \显然有
|\vec{a}|^{2}|\vec{b}|^{2} \geqslant(\vec{a} \cdot \vec{b})^{2}\|\vec{a}|^{2}=a_{1}^{2}+a_{2}^{2}+\cdots+a_{n}^{2}，|\vec{b}|^{2}=b_{1}^{2}+b_{2}^{2}+\cdots+b_{n}^{2}\(\vec{a} \cdot \vec{b})^{2}=\left(a_{1} b_{1}+a_{2} b_{2}+\cdots+a_{n} b_{n}\right)^{2}，即证\其中取等号的条件，就是
\end{array}
$$
　　柯西不等式的简单记忆方式就是：平方之和的积≥积之和的平方。

柯西施瓦茨不等式

$$
\int_{a}^{b} f^{2}(x) d x \cdot \int_{a}^{b} g^{2}(x) d x \geqslant\left(\int_{a}^{b} f(x) g(x) d x\right)^{2}
$$

　　柯西-施瓦茨不等式是在同济课本里出现过的，虽然在习题里面，但也是非常重要的不等式。这个不等式和柯西不等式时异曲同工的，积分可以看成是和，那左边依然是平方和的积，右边依然是积之和的平方。

权方和不等式

　　权方和不等式是柯西不等式的一个推论。
$$
\frac{a^{2}}{x}+\frac{b^{2}}{y} \geqslant \frac{(a+b)^{2}}{x+y}
$$

常见函数不等式

$\sin x<x<\tan x,x \in\left[0, \frac{\pi}{2}\right]$

$e^{x}-1 \geqslant x, x \in R$，注意，当x为0的时候，等号成立，当x＜0的时候，这个不等式也是成立的。

$\frac{x}{1+x}<\ln (1+x)<x, \quad x>-1$
